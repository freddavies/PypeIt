Functionality/Performance Improvements and Additions
----------------------------------------------------

- Added the possibility to use ``pypeit_sensfunc`` to compute a sensitivity function
  using the the 1D spectrum generated by ``pypeit_coadd_1dspec``, i.e., the OneSpec datamodel.
  Note that this is only possible for non-echelle data.

- Added a new class to interface with the atmospheric extinction data.
